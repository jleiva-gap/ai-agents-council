# Council Identity

AI Council uses a fixed naming layer to keep the UX, prompts, and logs consistent.

| Function | Identity | Log Example |
| --- | --- | --- |
| Proposal | Axiom | `[ AXIOM ] Parsing input...` |
| Critique | Sentinel | `[ SENTINEL ] Challenging assumptions...` |
| Refinement | Forge | `[ FORGE ] Refining execution details...` |
| Synthesis | Vector | `[ VECTOR ] Generating consensus...` |
| Validation | Collective | `[ COLLECTIVE ] Validating the converged result...` |

## Where It Appears

- prompt headers
- `session/deliberation-plan.json`
- `session/council-plan.json`
- `session/visual-reference.md`
- `logs/session.log`
- interactive shell legend
